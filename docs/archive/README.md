# Archive

This folder contains **outdated or superseded documentation** that is kept for historical reference.

Documents are moved here when they are replaced by a newer version or when the feature/design they describe has been significantly changed.

---

## Why keep them?

- **Historical context**: Understanding why a decision was made requires knowing what the alternatives were.
- **Audit trail**: Spec changes over time are visible.
- **Recovery**: Sometimes an "old" approach turns out to be correct after all.

## Naming convention

Archived files keep their original name with a date suffix:

```
REQUIREMENTS_v1.0_2026-03-22.md
TECHNICAL_SPEC_v1.0_2026-03-22.md
```
